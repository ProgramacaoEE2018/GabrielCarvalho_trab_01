Programa de cálculo de nota necessária na VF:

Primeiramente o programa pede ao usuário quantas VE's foram feitas. Após isso, de acordo com o número
de VE's escolhido, o programa pede a nota de cada uma delas. EM seguida, é pedida a nota de VC. A 
partir daí o programa calcula o quanto é preciso tirar na VF sendo que se a nota necessária for menor
do que 4, ele exibe 4 e se for maior, ele exibe a própria nota calculada. Por fim, o programa pergunta
se o usuário deseja fazer essa conta para outra matéria, repetindo o processo descrito acima.
